# BrewHub Monorepo

Monorepo de Brewhub, prueba práctica de un login con back-end en PHP + Laravel y front-end en ReactJS + Tailwind CSS.

## Requerimientos

### Backend

- PHP v8.2.x o superior
- Composer 2.6.x o superior
- MySQL

### Frontend

- NodeJS v20 o superior
- npm v10 o superior
